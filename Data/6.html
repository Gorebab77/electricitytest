4.2.3.2 기타 설정 항목 (006)

IDMTSW2_3_2
대상
■1P2W 단독 , ■1P2W 변성기부 
■3P4W 단독 , ■3P4W 변성기부
항목▶LP기록 (연계시험 4.5.1, 4.5.2)
▶통신속도 변경
▶부가신호 장치 출력선택 (연계시험
   5.2.3.1, 5.2.3.2, 5.2.3.3)
▶경부하 오결선 감지 체크
▶실시간 LCD표시값 통신출력 
   (연계시험 4.3.9.2)
▶과부하 전류 차단 설정/해제
▶단순표시 및 무부하 시 부하동작표시
   설정(연계시험 5.1.1.2)
규정
▶정전등 이상동작으로 인혜 계기가
   재부팅될때 이전속도 설정 값 유지
▶계기는 최대전류(변성기부 계기의 경우
   정격전류)의 10% 이상에서만 오결선을
   검출할 수 있어야 한다.
▶과부하 전류 차단 기능이 OC-E로 설정될
   경우, 과부하 전류 차단 동작조건 : 전류
   계측값이 최대 전류(변성기부 계기의
   경우 최대전류는 정격전류의 1.2배)의
   1.2배 이사이고 온도 측정값이 +85℃
   이상인경우로서, 두가지 조건이 모두
   충족된 시점부터 1시간 이상 지속될 
   경우 자동으로 부하차단하여야 한다.
   - 통신 : False = OC-E과부하전류 차단
   설정, TRUE = 0C-D 과부하전류 해제
   ※S/W 과부하 전류 차단기능의 동작
   시험은 H/W 시험에서 검토하기로함.
▶무후하 시 부하동작 표시설정 시 무부하
   시 동작 상한 표시는 1 상한 방향을 가리
   키며 12시방향부터 0.5초간격으로 회전.
   LCD는 현재 날짜,시간만 순환표시
환경
▶피시험계기(AMIG 전력량계)
▶STT(기능시험 자동화 장치) 
조건
(전처리)
▶부하 투입 조건①
   - 1P2W - 단독계기 - 60(5)  : 
   → T, 220V, 5.8A, 60Hz, 120도
   - 1P2W - 단독계기 - 120(10) : 
   → T, 220V, 11.8A, 60Hz, 120도

   - 1P2W - 변성기부계기- 5(2.5) :
   → T, 220V, 0.48A, 60Hz, 120도

   - 3P4W - 단독계기 - 120(10) : 
   → R, 220V, 11.8A, 60Hz, 120도
   → S, 220V, 11.8A, 60Hz, 120도
   → T, 220V, 11.8A, 60Hz, 120도

   - 3P4W - 변성기부계기 - 5(2.5) : 
   → R, 110V, 0.48A, 60Hz, 120도
   → S, 110V, 0.48A, 60Hz, 120도
   → T, 110V, 0.48A, 60Hz, 120도

▶부하 투입 조건②
   - 1P2W - 단독계기 - 60(5)  :

   - 1P2W - 단독계기 - 120(10) : 
   → T, 220V, 12.2A, 60Hz, 120도

   - 1P2W - 변성기부계기- 5(2.5) :
   → T, 220V, 0.52A, 60Hz, 120도

   - 3P4W - 단독계기 - 120(10) : 
   → R, 220V, 12.2A, 60Hz, 120도
   → S, 220V, 12.2A, 60Hz, 120도
   → T, 220V, 12.2A, 60Hz, 120도

   - 3P4W - 변성기부계기 - 5(2.5) : 
   → R, 110V, 0.52A, 60Hz, 120도
   → S, 110V, 0.52A, 60Hz, 120도
   → T, 110V, 0.52A, 60Hz, 120도
