방법
▶시험1 : Data Overflow 및 Selective
   Access : 
   - 전력량계에 전원을 투입한다.
   - LP 기록주기 “1분”으로 변경.
   - 부하투입 조건① 투입
   - 400초 대기
   - Selective Access(마지막5개)확인
   - 총 LP발생 횟수 증가 확인
   - 평균전압전류 발생 횟수 증가 확인
   - LCD확인(육안) : 모든 자리가 1회전 한 
   후에는 “0”으로 복귀 후 재 누산
   (예:99999.9 → 00000.0) 
▶시험2 : 조회속도 확인
   - 전체 데이터 읽기 시간 확인 
     ①전체 데이터(검침데이터 및 LP)읽기
     ②소프트웨어 업데이트
